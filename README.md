## META_007 - 🦠 16S Amplicon Sequencing: MiSeq vs NextSeq

🧑‍💻 **User:** Genomics Platform

📑 **Context/Description:** Comparison of 16S rRNA amplicon sequencing performance between Illumina MiSeq and NextSeq platforms, using the same set of samples processed on both instruments.

🎯 **Goal:** Evaluate concordance and platform-specific differences in taxonomic profiling, diversity metrics, and overall sequencing quality between MiSeq and NextSeq outputs.

🔢 **Data:** Paired 16S amplicon sequencing runs of the same samples on MiSeq and NextSeq.
